Ementa

A disciplina aborda as condições básicas para o desenvolvimento Web, mediante a compreensão, domínio e aplicação das técnicas inerentes ao desenvolvimento de Websites. Compreendendo o estudo do desenvolvimento de páginas Web dinâmicas através das Linguagens HTML, CSS e JavaScript e o auxílio de bibliotecas e frameworks, propiciando ao estudante uma compreensão global do funcionamento de um sistema Web.
